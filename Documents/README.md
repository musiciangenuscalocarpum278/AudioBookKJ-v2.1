# Documents - AudioBook-KJ V2 Workflow Guide

This directory contains the user manuals and workflow guides for the AudioBook-KJ V2 web application.

### Purpose of these documents:
- Explain the overall workflow of the web application.
- Provide detailed instructions for each view: Audio Studio, Video Studio, and Post-Production.
- Clarify the order of operations to prevent confusion between audio, video, timeline, and export processes.
- Serve as onboarding documentation for new users, developers, or AI agents taking over the project.

## Document List

- [01-overview-webapp-workflow.md](01-overview-webapp-workflow.md) - Overall workflow from script to final video export.
- [02-audio-studio-workflow.md](02-audio-studio-workflow.md) - Guide for creating scripts, voice casting, rendering audio, and adding to the timeline.
- [03-video-studio-workflow.md](03-video-studio-workflow.md) - Guide for managing assets, creating scenes, generating frames/videos, and syncing them to the timeline.
- [04-post-production-workflow.md](04-post-production-workflow.md) - Guide for timeline synchronization, previewing, mixing, and exporting the final audio/video.
- [05-project-media-workflow.md](05-project-media-workflow.md) - Guide for managing projects, media files, and project folders.
- [06-agent-handoff-workflow.md](06-agent-handoff-workflow.md) - Guidelines for AI agents (Claude/Gemini/Codex) or developers to resume work without conflicting with others.
